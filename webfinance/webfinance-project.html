About this project

Projet R is a complete financial management dashboard built for a web agency of 4 partners. It centralizes all financial data — income, expenses, client tracking and analytics — in a clean, real-time interface with no backend required.

Each team member's contribution is tracked individually. The client directory automatically builds from income entries, with a full payment lifecycle: 30% deposit, 70% balance at delivery, or single 100% payment — all visualized with progress bars and status indicators.

  • Real-time KPI dashboard — net revenue, income, expenses, receivables
  • Per-member contribution tracking with percentage breakdown
  • Client directory auto-built from transactions with full payment history
  • 30% deposit / 70% balance workflow with one-click delivery validation
  • 6 interactive Chart.js charts — pie, bar and monthly trend line
  • Password-protected edit & delete, CSV export, FR/EN toggle
