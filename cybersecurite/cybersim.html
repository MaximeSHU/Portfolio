About this project

CyberSim is a fully local, interactive web simulation designed to raise awareness about online dangers. Users are immersed in a realistic digital environment — a smartphone or desktop — and face real-world cyber threats without any actual risk.

At the end of each simulation, a detailed report reveals every attack the user fell for, along with personalized security recommendations. A pedagogical explanations page then breaks down each threat and how to avoid it.

  • Phishing — fake Amazon email & fake bank SMS
  • Credential theft via fake Instagram contest
  • Grooming — social engineering DM scenario
  • Man-in-the-Middle via unsecured Wi-Fi network
  • Malware install simulation (JeuGratuit.exe)
  • Webcam spyware — real camera access request
  • Auto-generated stolen data report (.txt)
